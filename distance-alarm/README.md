# Distance Alarm

A device that generates a warning sound when an object moves within a certain range.

## Components

### Hardware

* Arduino UNO
* Ultrasonic Sensor HC-SR04
* Passive Piezo-Buzzer KY-006

### Software

* IDE: Arduino IDE
