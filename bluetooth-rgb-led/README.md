# Bluetooth RGB LED

Control RGB LED from Android app via Bluetooth serial communication.

## Components

### Hardware

* Arduino UNO
* 4-Pin RGB LED Module
* Bluetooth Module HC-05

### Software

* IDE: Arduino IDE
* Android app: Bluetooth Terminal HC-05
